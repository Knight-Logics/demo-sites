# Demo Sites

Public repository for lightweight pitch demos and concept sites.

## Current demos

- `titan-roofing-repairs/` — owner-led Georgia roofing demo concept built from supplied branding and project images

## Publishing model

- Root `index.html` acts as the public landing page for all demos.
- Each demo lives in its own subfolder with isolated assets.
- Public-safe details only: direct phone, email, legal, and warranty data stay out of published demos unless approved.
